★ Recomendação final

Pair 1 — Fraunces + Inter Tight + JetBrains Mono

Fraunces é o único variable + optical sizing deste grupo. Significa que uma única fonte cobre desde o hero a 48px (com personalidade dramática) ao mini-card a 22px e até subtítulos a 18px (com desenho adaptado para legibilidade) — sem precisares de uma família secundária. Tem o axis SOFT para tornar as curvas mais humanas e WONK para swashes irregulares (use it sparingly). Cobre PT-PT, EN, FR sem problemas.

Inter Tight resolve o problema de Syne ser uma display font a fazer de UI font. É a default escolha de Linear, Vercel, e outros produtos editoriais modernos — desaparece de propósito para deixar o conteúdo respirar.

JetBrains Mono substitui DM Mono com mais pesos (8 vs 2), mais glifos, e ligaturas opcionais — útil para hierarquia de metadados, prefixos de email, contagens.

Performance · total ~180KB woff2 (subset latin-ext)
License · SIL Open Font · todos free para uso comercial
